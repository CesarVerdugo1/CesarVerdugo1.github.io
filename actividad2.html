What is HTML?

HTML es como el esqueleto de una pagina web. Es la forma de dar instrucciones a la pagina

Explain the head and the body tags.

El head es la parte de la página que no se ve, pero que le dice cosas importantes a la página y al navegador.

Explain the section, article, aside, header, footer tag?

Section

Es una sección de la página que agrupa contenido relacionado.

Article

Es un bloque de contenido independiente que puede ser distribuido por sí mismo.

Aside

Es un bloque de contenido que está relacionado con el contenido principal, pero que se puede considerar como un complemento o una información adicional.

Header

Es la parte superior de una página o sección que generalmente contiene el título, el logo y la navegación.

Footer

Es la parte inferior de una página o sección que generalmente contiene información de contacto, derechos de autor y enlaces relacionados.

    What is CSS?

    CSS es una hoja de estilo que se usa para dar formato a una página web. Es como la ropa que se pone sobre el esqueleto HTML.

    What is the purpose of CSS?

    El propósito de CSS es controlar la apariencia de una página web, como los colores, las fuentes, los márgenes y la disposición de los elementos.

    How do you apply a CSS to an HTML (explain the 3 options)

    1. Inline: Se aplica directamente al elemento HTML usando el atributo style.
    2. Internal: Se define dentro de la etiqueta; en el head del documento HTML.
    3. External: Se define en un archivo separado con extensión .css y se enlaza al documento HTML usando la etiqueta link.

    Tag Selector

    Aplica estilos a todas las etiquetas del mismo tipo, por ejemplo todos los párrafos.

    Class Selector

    Aplica estilos a todos los elementos que tengan una misma clase. Puede usarse en varios elementos.

    Tag and Class Selector

    Aplica estilos solo a una etiqueta específica que tenga cierta clase. Es más específico.

    Id Selector

    Aplica estilos a un único elemento con un id específico. No debe repetirse.

    ¿Qué es una class?

    Es un atributo que permite compartir estilos entre varios elementos.

    ¿Qué es un id?

    Es un identificador único para un solo elemento dentro de la página.

    Relación con CSS

    CSS usa clases e ids como selectores para aplicar estilos a los elementos HTML.

    CSS Box Model

    Todo elemento es una caja compuesta por contenido, padding, border y margin. El tamaño total incluye todos estos elementos.

    Media Queries

    Permiten cambiar estilos según el tamaño de la pantalla para crear diseño responsivo.

    display: grid

    Organiza elementos en filas y columnas al mismo tiempo (2 dimensiones).

    display: flex

    Organiza elementos en una sola dirección, fila o columna, y facilita la alineación.
